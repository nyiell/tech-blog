üéØ Summary

What Jekyll Does:

  • Reads your _posts/*.md files
  • Converts Markdown ‚Üí HTML
  • Applies Chirpy theme CSS/JS
  • Generates sidebar navigation
  • Creates tag & category pages
  • Builds table of contents
  • Adds syntax highlighting to code
  • Calculates reading time
  • Outputs everything to _site/ folder

Result: A fast, beautiful, responsive blog with dark mode, search, and all modern features!
